Frequently Asked Questions

How does FileCryp ensure file security?
FileCryp employs advanced encryption algorithms and decentralized storage on the blockchain to guarantee the utmost security for your files.
What is IPFS, and why is it used?
IPFS is a decentralized protocol for secure and efficient storage and sharing of files in FileCryp.
Can I access my files later?
Yes, users can decrypt their files anytime using provided keys as long as IPFS nodes have the content available.
What file types are supported?
Images, documents, archives, and many file types are supported for compression and upload.
How does file compression work?
Files are compressed using efficient algorithms before encryption and upload to reduce size and improve speed.
